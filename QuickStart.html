Quick Start On Exploring Transcriptomics Datasets

  1. Load transcriptomics tab. Choose a dataset to explore from the menu and click Load Dataset.
    The dataset appears showing some annotation data in the first columns and then the log2 normalised gene expression values. Datasets have expression values of individual spots as rows (mapping to ProbeNames (nucleic acid sequences) and genes - one ProbeName may map to several spots, and one Gene may map to several ProbeNames). Each column is the mean expression for one vaccine~day combinations, where participants have been grouped by treatment (e.g. BOOSTRIX_1 column is group mean for Boostrix vaccinated participants on Day 1). Both Fold Change (from Day 0) and Raw Expression value datasets are available.
  2. Explore By Gene tab: Select this and the following options appear:
    1. Select Genes: Filter the rows of spot expression values by some criteria. If you click Apply Filters with the default setting the 10 rows with spots having the highest value in the column matching the Vaccine~Day combination selected in the menus will be selected and all other rows filtered-out.
      Select Gene Averages to select top 10 genes based on the mean gene expression values (calculated from all spots mapping to probes that map to that gene). instead of spots.  As returning more than 100 rows from a filter will take a long time to display, at the top right is a box to enter a maximum number of rows to return whatever the filter settings - the default is 100.
      Other filter options include keyword to select rows using a text-based filter on the annotation columns; and by value of the rows in the selected column or across all columns. See the detailed instructions for further information.

      Once you have clicked "Apply Filters" the following tabs now appear:
    2. Selected Genes  lists the selected rows (spots or genes) that meet the filter criteria.
    3. Genes:Series: plot the time course of selected rows values. Select some vaccines and days to plot, and the spots / genes to plot. Click Plot button each time you change options.
    4. Genes->Modules lists genes matching selected rows, and identifies which modules contain these genes (if any). Filtered spots/genes that do not match any module are not listed - pay attention to the Rank column as it assists with understanding which rows are missing because they do not match any modules.
    5. Modules box plot of the all the row values which map to genes contained within the selected modules. Include the filters-selected genes as a separate "pseudo-module". Plots automatically after every options change. Note: the values are from the treatment-time combination selected in the menus in Select Genes - to see a different time or treatment you must change the filters and re-apply.
    6. Module->Genes  For every selected module it lists the genes contained within that module. Select a module from the menu to plot a box plot summarising the row values for each gene (some genes may have only one spot, others many probes/spots). Genes that also passed the filters indicated by ▶︎. Plots automatically. Some modules may have very few filtered genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to the vaccines.
    7. Module:Series plot the time course of row values within each module. Select some vaccines and days, and some modules from those identified by the genes filter. Click Plot button to create or update plot. Include the selected genes as a separate "pseudo-module".
  3. Explore By Module tab:
    This works on the same principles as Explore By Gene except you filter modules by keyword or based on the mean or median value of all the spots mapping to that module.
    1. Select Module filter modules by mean or median spot values, or keyword. Click Apply Selection.
      Once you have clicked "Apply Filters" the following tabs now appear:
    2. Selected Modules - box plot of the all the row values which map to genes contained within the selected modules. Plots automatically.
    3. Module->Genes For every selected module it lists the genes contained within that module and then plots a box plot summarising the row values for each gene (some genes may have only one spot, others many probes/spots). Select a module from the menu. Plots automatically. Some modules may have very few upregulated genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to the vaccines. Note: the values are from the treatment-time combination selected in the menus in Select Module - to see a different time or treatment you must change the filters and re-apply.
    4. Module:Series plot the time course of row values within each module. Select some vaccines and days, and some modules. Click Plot button each time you change options. Select modules to plot from the options: modules selected in Select Modules tab; all modules available in database, all modules available in database clustered under their functional Title categories.
