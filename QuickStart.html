Quick Start On Exploring Transcriptomics Datasets

  1. Load transcriptomics tab. This now has only one consolidated dataset, so just click Load Dataset.
    • The dataset appears showing some annotation data in the first columns and then the log2 normalised gene expression (GE) values for each Treatment~Time combination.
    • The "Gene" column is a portmanteau label in which I have tried to compensate for variable annotation of the chipset. See details for further info.
    •  Datasets now have expression values of individual ProbeNames as rows (mapping to the mean of any duplicated spots on the chip with the same ProbeName (nucleic acid sequences)). Therefore the ProbeNames now are unique and not duplicated. This was a compromise to allow integration of 306B expression values which were run on an Agilent chip which has only a subset of Probes from the BIOVACSAFE chip.
    • Subsequently if you choose to work with gene averages, resulting datasets will have one gene per row.
    • Each GE column is the mean expression for one treatment~day combinations, where participants have been grouped by treatment (e.g. BOOSTRIX_1 column is group mean for Boostrix vaccinated participants on Day 1).
  2. Explore By Gene tab: Once a dataset is loaded this tab appears. Select this and the following options appear:
    1. Select Genes: Filter the rows of probe expression values by some criteria.
      • You must select a treatment and time (I suggest try Fluad C, day 1 to start)
      • If you click Apply Filters with the default setting the 10 rows with probes having the highest value in the column matching the Vaccine~Day combination selected in the menus will be selected and all other rows filtered-out.
      • Select Gene Averages to select based on the mean gene expression values (calculated from all  probes that map to that gene).
      • As returning more than 100 rows from a filter will take a long time to display, at the top right is a box to enter a maximum number of rows to return whatever the filter settings - the default is 100.
      • Other filter options include keyword to select rows using a text-based filter on the annotation columns; and by value of the rows in the selected column or across all columns. See the detailed instructions for further information.
      Once you have clicked "Apply Filters" the following tabs now appear:
    2. Selected Genes  lists the selected rows (probes or genes) that meet the filter criteria.
      • Use the red ⤓Text File button to download a useful text file with all the annotation symbols for the selected probes/genes. Copy-paste the Gene labels into boxes in this tool that accept identifiers. For other online resources such as Gene Ontology or other analysis tools use the GeneSymbol or Entrez or ENSEMBL IDs.
    3. Genes:Series: plot the time course of selected probes/genes values. Select some vaccines and days to plot, and the probes / genes to plot. Click Plot button each time you change options.
    4. Genes->Modules Modules work on the basis of the GeneSymbol name. The table lists selected genes (or genes matching selected probes), and identifies which modules contain these genes (if any). Filtered genes that do not match any module are not listed - pay attention to the Rank column as it assists with understanding which selected genes are missing because they do not match any modules.
    5. Modules box plot of the all the mean of probe values which map to genes contained within the selected modules (remember, many spots -> one probe, many probes -> one gene). You can include the filters-selected genes as a separate "pseudo-module" to compare the fold-change in GE of the selected genes compared with the module means. Plots automatically after every options change. Note: the box plot values are for the treatment-time combination selected in the menus in Select Genes - to see a different time or treatment you must change the filters and re-apply.
    6. Module->Genes  For every selected module it lists all the genes contained within that module. Select a module from the menu to plot a box plot summarising the row values for each gene (some genes may have only one probe, others many probes/spots). Genes that are also included by the filters indicated by ▶︎. Plots automatically. Some modules may have very few filtered genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to the vaccines.
    7. Module:Series plot the time course of row values within each module. Select some vaccines and days, and some modules from those identified by the genes filter. Click Plot button to create or update plot. Include the selected genes as a separate "pseudo-module".
    8. CRC305E. This trial included participants receiving Fendrix, Fluad or saline placebo injection into the leg, and then blood samples on day 0 and one timepoint post vaccination, and a muscle biopsy at the single post-vaccination time point from both injected and uninjected legs. GE values are presented as fold change in blood GE from day 0 to the single post immunisation time point, and fold change in muscle GE between injected:uninjected legs on the single post immunisation time point. As there were only 3 participants per time point, some variation may exist in the mean GE values that can be plotted in the time course graphs. Therefore, this tab allows the GE of individual participants to be visualised. Select a treatment~tissue~time (in hours) combination and the plot of participants and the means/medians will be shown for selected genes. The genes selected by the filters in Select Genes tab are filled-in automatically, but any Gene identifier can be pasted in.
  3. Explore By Module tab:
    This works on the same principles as Explore By Gene except you filter modules by keyword or based on the mean or median value of all the probes mapping to that module (modules->genes->probes). Probes are not averaged into gene values.
    1. Select Module filter modules by mean or median probe values, or keyword. Click Apply Selection.
      Once you have clicked "Apply Filters" the following tabs now appear:
    2. Selected Modules - box plot of the all the row values which map to genes contained within the selected modules. Plots automatically.
    3. Module->Genes For every selected module it lists the genes contained within that module and then plots a box plot summarising the row values for each gene (some genes may have only one probe, others many probes). Select a module from the menu. Plots automatically. Some modules may have very few upregulated genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to the vaccines. Note: the values are from the treatment-time combination selected in the menus in Select Module - to see a different time or treatment you must change the filters and re-apply.
    4. Module:Series plot the time course of row values within each module. Select some vaccines and days, and some modules. Click Plot button each time you change options. Select modules to plot from the options: modules selected in Select Modules tab; all modules available in database, all modules available in database clustered under their functional Title categories.
  4. Network Genes
    This is a stand-alone module. Plots Venn / Euler / UpsetR diagrams or network graph of genes in common or unique to specific vaccine~day combinations, based on gene expression fold-change or gene rank values. Note no statistical analysis is done.
    1. Select vaccine~day combinations and Add to the list to plot.
    2. Select number of genes to include, and whether top N selected on basis of descending mean gene expression value (highest genes selected) or reverse (lowest N genes selected).
    3. Click Plot to draw graph, and after any change to selections.
    4. Select Network, Venn, Euler or UpsetR plot.
    5. The plots automatically redraw when you apply subsequent filters to include / exclude genes based on number of connections to vaccines or value/rank.
    6. Data tables list genes based on the filters and will change automatically. Export as single Excel file with sheets for each table. Export plots as High Res PNG by clicking button.
    7. See detailed instructions for further information on filtering and plots.
