Quick Start On Exploring Transcriptomics Datsets

  1. Enter the password if using the app on shinyapps.io server. Click Enter
  2. Load transcriptomics tab. Choose a dataset to explore from the menu and click Load Dataset. Dataset has individual probes (mapping to genes) in rows and vaccine~day combinations in columns (group mean). Fold Increase and Raw Expression values available.
  3. Explore By Probe tab:
    1. Select Probes tab: select a vaccine and a day to sort the probes by. Click Apply Filters. Top 10 probes for selected vaccine on selected day are selected. Select Gene Averages to select top 10 genes instead of probes. Use other search filters to select genes / probes.
      The following tabs now appear:
      1. Selected Probes tab lists the selected probes / genes with descriptions for probes.
      2. Probes:Series tab: Select some vaccines from Treatment menu, and Days. Click Plot button. Kinetics of selected probes appears. Click Plot button each time you change options.
      3. Genes->Modules tab lists the genes matching selected probes or genes and which mapped modules contain these genes (if any).
      4. Modules tab summarises the probe values within each mapped module that map the selected probes / genes. Include the selected genes as a "pseudo-module". Plots automatically.
      5. Module->Genes tab summarises the probe values for genes within each mapped module, one by one. Select a module from the menu and the probe values summarised as boxplots for each gene. Plots automatically. Some modules may have very few upregulated genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to vaccines.
      6. Module:Series tab allows the kinetics of the probe values within each module to be plotted over time. Select some vaccines from Treatment menu, and Days. Click Plot button to create plot. Click Plot button each time you change options. Include the selected genes as a "pseudo-module".
  4. Explore By Module tab:
    1. Select Module tab: select a vaccine and a day to sort the tmod modules mean or median probe values by. Click Apply Selection. Top 10 modules for selected vaccine on selected day are selected. Use other search filters to select modules.
      The following tabs now appear:
      1. Selected Modules tab summarises the probe values within each selected module. Plots automatically.
      2. Module->Genes tab summarises the probe values for genes within each selected module, one by one. Select a module from the menu and the probe values summarised as boxplots for each gene. Plots automatically. Some modules may have very few upregulated genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to vaccines.
      3. Module:Series tab allows the kinetics of the probe values within each selected module to be plotted over time. Select some vaccines from Treatment menu, and Days. Click Plot button to create plot. Click Plot button each time you change options. Select modules to plot from the options: modules selected in Select Modules tab; all modules available in database, all modules available in database clustered under their functional Title categories.
  5. For help on Lookup, Cells and Cytokines, and for more detailed information: see Instructions For Use tab in ReadMe.
