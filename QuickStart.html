Quick Start On Exploring Transcriptomics Datasets

  1. Enter the password if using the app on shinyapps.io server. Click Enter
  2. Load transcriptomics tab. Choose a dataset to explore from the menu and click Load Dataset. Dataset has individual spots in rows (mapping to ProbeNames (nucleic acid sequences) and genes - one ProbeName may map to several spots, and one Gene may map to several ProbeNames) and vaccine~day combinations in columns (group mean). Fold Increase and Raw Expression values are available.
  3. Explore By Spot tab:
    1. Select Spots tab: select a vaccine and a day to sort the spots by. If you click Apply Filters with the default setting the top 10 spots for selected vaccine on selected day are selected. Select Gene Averages to select top 10 genes instead of spots. Apply other search filters to select genes / spots.
      The following tabs now appear:
      1. Selected Spots tab lists the selected spots / genes with descriptions for spots.
      2. Spots:Series tab: Select some vaccines from Treatment menu, and Days. Click Plot button. Kinetics of selected spots appears. Click Plot button each time you change options.
      3. Genes->Modules tab lists the genes matching selected spots / genes, and which mapped modules contain these genes (if any).
      4. Modules tab summarises the spot values within each mapped module that map the selected spots / genes. Include the selected genes as a "pseudo-module". Plots automatically.
      5. Module->Genes tab summarises the spot values for genes within each mapped module, one by one. Select a module from the menu and the spot values summarised as boxplots for each gene. Plots automatically. Some modules may have very few upregulated genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to vaccines.
      6. Module:Series tab allows the kinetics of the spot values within each module to be plotted over time. Select some vaccines from Treatment menu, and Days. Click Plot button to create plot. Click Plot button each time you change options. Include the selected genes as a "pseudo-module".
  4. Explore By Module tab:
    1. Select Module tab: select a vaccine ~ day combination by which to sort the tmod modules mean or median spot values. Click Apply Selection. Top 10 modules for selected vaccine on selected day are selected. Use other search filters to select modules.
      The following tabs now appear:
      1. Selected Modules tab summarises the spot values within each selected module. Plots automatically.
      2. Module->Genes tab summarises the spot values for genes within each selected module, one by one. Select a module from the menu and the spot values summarised as boxplots for each gene. Plots automatically. Some modules may have very few upregulated genes, or a wide spread of gene expressions. Reflects how the clustering of genes within a module may not reflect the clustering of genes in response to vaccines.
      3. Module:Series tab allows the kinetics of the spot values within each selected module to be plotted over time. Select some vaccines from Treatment menu, and Days. Click Plot button to create plot. Click Plot button each time you change options. Select modules to plot from the options: modules selected in Select Modules tab; all modules available in database, all modules available in database clustered under their functional Title categories.
  5. For help on Lookup, Cells and Cytokines, and for more detailed information: see Instructions For Use tab in ReadMe.
