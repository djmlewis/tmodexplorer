Instructions For Use

Password

When running on the shinyapps.io server the software requires a password. This is to protect the datasets. Please contact d.j.lewis@surrey.ac.uk to be given the password. Enter the password and click the button to unlock the menus. When running locally no password is required.

Loading Transcriptomics Data

You must load a transcriptomics dataset before you can do anything with transcriptomics data. The transcriptomics datasets are already uploaded to the server. Select a dataset. Click Load. All existing plots and data tables will be cleared. Once you click "Load" the name of the data set will appear below the tabs, and two more tabs will appear:
  • Explore By Probe
  • Explore By Module

Explore Transcriptomics By Probe tab

Select this tab to explore the datasets using individual probes in the array. The idea is to choose a time point and a treatment, and then sort the probes based on their values (gene expression or fold-increase) in that combination (one column of the dataset), and then additionally filter by keyword search or value. Note - if you include more that 200 rows then plots are slow to draw. Do not click "Plot" buttons or "Apply" again as the cycle just repeats!

Select tab - probes

You must click the Apply Selections button before any filters or selections are applied, and again after any changes. This is to speed-up calculations. Once you click Apply Selections, several other tabs will appear to allow visualisation of the genes and modules. The search terms you included appear below the tabs.

You must select at least one of the numbered filters. Filters are applied one after another, left to right - first the keyword regex search, then the value restriction, finally which rows to include. Probably you will always use Sorted Column Row Numbers to restrict the number of sorted probes, possibly alone or in combination with either regex or Sorted Column Values. Never, ever try to search with no filters selected - it will return all 60k+ probes, very slowly, and I have set a block on this.

1. Regex - keyword search

"Regex" is a keyword, free-text search that uses regex, with some limitations. Multiple terms can be entered into the box separated by commas and are OR'd (search returns as many of those terms as it can find, treating each term separately). Commas are interpreted as search term dividers and cannot be 'escaped'. Select whether to search gene description or gene name, or probe ID ("Probe") or ProbeName. Probe is least useful as they are not very standardised between datasets.

Spaces will be stripped-out automatically for you in Gene and Probe name searches, so you can easily paste in lists copied from tables. For Description searches the spaces are left in so beware: regex is "literal" so if you type "B cell" it is going to look for "B cell" - including the "".

You can enter partial terms e.g. interferon finds interferon and interferons, as regex looks for those characters anywhere. If you select "Whole Word" then interferon will only match interferon. Whole Word is important when searching for genes and probes as "CCL2" will find CCL2, CCL27 etc unless Whole Word is selected.
Upper/lower case is ignored.
Beware including special regex characters by mistake.

Regex search  ignores the "Selected Column" and any values (see below) - it just finds rows in the data set that match the search. However, those rows will be sorted in order of the value in the "Selected Column", and filtered on these values. You may want to switch off the Row Numbers Within Range when using regex as it may reduce the number of free-text hits you would have found. However, beware returning hundreds of rows as the app will freeze for a long time. Consider setting some Values Within Range limits.

You can also paste saved search lists into the regex box that you downloaded from the Selected Probes tab, or copied from one of the Lookup tables - be sure to match Gene and Probe / Name. This allows fairly easy repetition of searches across datasets, at least for Genes.

2. Sorted Column Values

Pick a  treatment ~ time combination you want to explore from the menus. This treatment~time "column" is the basic unit for tmodExplorer - it combines a treatment and a time point, e.g. "Boostrix_7" contains mean values on day 7 for participants who received Boostrix. All data sets are based on these columns. Time series plots will indicate the sorted column with ▼▲.

Sort descending for highest values in top rows.
Select Gene Averages to have expressions of individual probes that map to a gene averaged into one row, otherwise it will be one probe per row and genes may repeat. This is important if you restrict by "Sorted Row Numbers": e.g. if you select 10 rows and have Gene Averages selected you may include many more than 10 probes, but if Gene Averages is not selected you just get the top 10 probes which may map to less than 10 genes. As the modules work in genes, it is probably better to select Gene Averages, unless you specifically want to follow distinct probes. A disadvantage of gene averages is that you lose the gene descriptions, which are probe-specific. Copy the Gene list on the Selected Probes tab and use  the Gene Lookup tab to lookup genes by their names and see the probe descriptions.

Max/min expression values will be pre-entered for you into the range boxes when selecting a column, using floor() and ceil() and so will deliberately exceed the actual values. Click Column to reset to max/min for selected column, or Data for max/min of whole dataset in all columns. Value restrictions apply only to values in the column selected for sort - so you will filter based on the values in the column you selected. This is important - these restrictions do not affect other treatment~time columns.

3. Sorted Column Row Numbers

This is probably the most useful restriction and it is probably a good idea to keep this selected, as over 200 rows is slow to calculate and plot. You can restrict rows to have top N or bottom N rows, or have a slice. If there are fewer rows available after the previous selections than you request you will get a cryptic warning. Row restrictions apply  to the sorted values in the column selected for sort - so you will filter based on the values in the column you selected.

Selected Probes tab

A data table showing the probes or genes (if averaged) that meet the filter settings is shown and can be downloaded as a csv file or as a PNG picture. The Vale shown is the one in the "Column" you selected. You may use the search boxes above and below the table to further restrict the rows displayed in the table, but this has no effect on other tabs or plots. Click "⤓ Gene List" or "⤓ Probe / ProbeName List" to save a comma-separated text file that can be copy-pasted into the regex search box to quickly repeat the search. Note a description appears when probes are displayed individually, and disappears if "Gene Averages" is selected on the Select tab. This is because descriptions are specific for probes.

Probes:Series tab

This allows the values of the probes or gene averages selected by the filters to be plotted for different treatment ~ time combinations, to visualise a time course. Use the slider to change plot height, and browser window width to change plot width. The selected treatment~time column is indicated by ▼▲ on the plot.

Treatment, Days and Gene/Probe menus

Click 'All' to enter all available, or select one by one from pop-down menu and they appear in the box. Click 'None' to clear. The order of the columns as they appear in the box is respected for all boxes like this - so the plot axis will retain the sequence you enter here, unless it is plotting as an XY scatter using numerical values. Use the cursor to move between entries to add or delete a column at that point. Select an entry and delete.

Plot button: click this to draw the plot, and again after changing any options.
Split: the data are split by treatment using facet_wrap.
Lines / Boxplot: XY graph of individual probes / genes, or a boxplot summarising probe/gene values.
0 |---- (zero) line: Place a dotted line at Y = 0 (baseline).
Points, Gridlines, Legend: show/hide.
SEM: Appears when Lines selected - show the SEM as a ribbon
Gene⫷Probes: Appears when Gene Averages selected in Select Probes tab, and Lines selected to plot. Instead of plotting a line for the average of the probe values mapping individual genes (±SEM), individual lines are plotted for the probes mapping to that gene - the lines are colour-coded to the gene, and different symbols are used. This helps to show the behaviour of different probes mapping the same gene. Hover over the points to identify the probe.

Genes->Modules tab

This panel lists the selected probes or genes and shows which - if any - modules contain that gene. Some genes you selected may not be included in any modules.

Modules tab

A boxplot summary of the values of the individual probes contained in the modules associated with the selected probes or genes is plotted automatically. Use the slider to change plot height, and browser window width to change plot width. If you change any of the criteria on the Select tab this plot will re-draw.

Show or hide the legend and 0 |---- (zero) line, or have ggplot2 graphics which are a bit nicer but slower. ggplot2 calculates the dimensions of the box slightly differently to base graphics.

Select "Include Selected As Module" and the genes selected by the filters can also be plotted as a "pseudo-module", for comparison with  the tmod modules.
"Module" plots one box per module, and "Title" groups the modules  by their functional / descriptive category titles which may result in fewer boxes.
The table gives the summary statistics and the number of probes mapped to the modules' genes (N).

Modules->Genes (Probes) tab

This expands the plot on the Modules tab  to allow an inspection of the modules, one by one. Use the slider to change plot height, and browser window width to change plot width.

Select a module from the menu to see the expression of its component genes. As each gene may have many probes a summary boxplot of the expression values of the probes mapping to the genes inside the selected module is plotted automatically. Note that a module may also contain genes that are not in the selection created by the Select tab. Genes within a module that are in your selection are denoted by the ︎► symbol next to the y axis, and in the data table. You may be surprised by how few genes in a module overlap with your selected genes.

Genes listed for the module but absent from the dataset rows, are labelled "Missing" on the plot, which can be toggled on-off.

Show or hide the legend and 0 |---- (zero) line , or use ggplot2 instead of base graphics.  ggplot2 uses a geom_boxplot with default values. Unlike the Selected Probes tab which shows the gene description in the table, the Module->Genes tab does not, as the descriptions refer to the probes, and here gene summaries are being shown.

Modules:Series (Probes) tab

While the Modules and Modules->Genes tabs only show the values of probes at one selected treatment ~ time combination, the Modules:Series tab allows the values of the associated modules to be plotted for all treatment ~ time combinations as a time course. To avoid delays when selecting options, nothing is plotted or re-plotted until the 'Plot' button is clicked. Use the slider to change plot height, and browser window width to change plot width.

Select treatment ~ time combinations from the menus. Select modules from the Modules menu which lists ones mapped to your selected probes or genes. Use All or None buttons to add all, or clear selections.

Boxplot summarises the values of probes within the module.
Lines plots a line graph of the mean value of probes in the modules.
SEM, Points and Gridlines appear when Lines is selected allowing an SEM ribbon,  points and gridlines to be plotted.
Split - uses facet_wrap to separate treatments into different panels.
Show / hide the legend and 0 |---- (zero) line.

Explore Transcriptomics By Module tab

The concept is similar to the Explore By Probe tab. However, here a data table is created in which the rows are sorted by the mean or median value of the modules as expressed in the selected treatment ~ time combination: e.g. the median expression of the probes within a given module on day 7 after Boostrix. It is important to note that tmod modules contain only a fraction of all the genes in the probeset, but this tab allows for an exploration of responses of the modules included in tmod.

Select modules tab

Like Select tab in Explore By Probe above, you select a treatment ~ time combination for which you want to sort the modules. Then filter modules using the filters (left to right) as described in Select tab for probes. In this case the sorting is based on the mean or median expression (which you can select) of the probes within the module. Sort descending has the highest expression values in top rows. See the Probe Select tab for description of how to use regex Keyword, Values Within Range and Row Numbers Within Range filters. Time series plots will indicate the sorted treatment~time column with ▼▲.

Regex searches either the Title (which is a description of the module, e.g. "plasma cells") or Module name (its designation e.g. DC.M4.11). Spaces will automatically be stripped from Module searches but retained for Title searches.

You may want to switch off the Sorted Column Row Numbers when using regex as it may reduce the number of hits you would have found. However, beware returning 600 modules as the app will freeze for a long time. Consider setting some Sorted Column Values limits instead - this will be based on values in the selected sort column. You can also paste saved search lists into the regex box that you downloaded from the Selected Modules tab - be sure to match Title and Module Name. This allows fairly easy repetition of searches across datasets.

You must click the Apply Selections button to perform the selection or after any changes to options. Clicking the button reveals Selected Modules,  Modules->Genes and Modules:Series tabs to visualise the modules.

Selected Modules (Modules) tab

This tab plots a boxplot summary of the probe expressions mapping the genes in the modules you selected. Use the slider to change plot height, and browser window width to change plot width.

Show / hide the legend and 0 |---- (zero) line, and  plot using ggplot2 which is nicer, slower and draws the boxes slightly differently to base graphics. You can colour the boxes as individual modules by selecting Module, or as groups by their functional / descriptive category using Title, which may result in fewer boxes when multiple modules are grouped under one title.

Modules->Genes (Modules) tab

This expands on the Selected Modules tab  to allow an inspection of the modules, one by one, looking at all the genes each module contains. It is essentially identical to Modules->Genes tab under Explore By Probe. Use the slider to change plot height, and browser window width to change plot width.

Select a module from the menu to plot the summarised expression of probes mapping to its component genes automatically.
Show or hide the legend and 0 |---- (zero) line, and  use ggplot2 instead of base graphics.  ggplot2 uses a geom_boxplot with default values.
Genes listed for the module but absent from the dataset rows, are labelled "Missing" on the plot, which can be toggled on-off.

Modules:Series (Modules) tab

This is very similar to the Modules:Series (Probes) tab as it allows the expression of the selected modules to be plotted for all treatment ~ time combinations as a time course. Use the slider to change plot height, and browser window width to change plot width. The selected treatment~time column is indicated by ▼▲ on the plot.

Select treatment ~ time combinations to include from the Treatment and Days menus.

Select which modules to plot using one of the menus underneath:
  • Modules You Selected With Filters  lists the modules displayed in the Selected Modules tab.
  • All Modules In Dataset lists all the modules available in this dataset by their names.
  • All Module Titles In Dataset menu lists all the available functional / descriptive module titles in this dataset - which group  modules into categories. 

Once you have selected some modules or titles in the  All Modules In Dataset or All Module Titles In Dataset you can save the list by clicking the download button that appears. To repeat this selection (e.g. on another dataset) you must paste this comma-separated text into the regex search box on the Select Modules tab. You cannot paste into these menus directly.

Click Plot button to create the plot and data table, and after any changes to module/treatment/day selections or options.

Boxplot plots a summary of the probe expressions in the modules
Lines  plots the module mean expression and reveals SEM to include an SEM ribbon, plus gridlines and points.
Split plots each treatment in its own panel
Legend and 0 |---- (zero) line can be shown or hidden.
The bars can be coloured by individual modules when Group By: Module is selected, or by module Title, which may result in fewer boxes.

Transcriptomics Lookup tab

Gene Lookup tab

Enter a gene, probe or ProbeName and click Lookup to see a table of matching genes. Select Whole Word to avoid matching multiple stems (e.g. "CCL2" would match CCL2, CCL20, CCL21, etc.). The search uses regex, and commas to separate multiple search terms in the same way as selecting probes by keyword: see that section for details. Spaces will always be stripped from this search as Gene / Probe  Names cannot have spaces.

You can paste the comma-separated lists downloaded by the various buttons on other tabs, to quickly describe the genes/probes.

Alternatively, leave the search box empty and click Lookup to list all 65k genes. The search boxes above / below the table can also be used to narrow the rows displayed.

Module Lookup tab

Enter a module name (e.g. DC.M4.1) or partial string (e.g. DC.M4) and click Lookup to see a table of matching modules. The search uses regex, and commas to separate multiple search terms in the same way as selecting probes by keyword: see that section for details.
Spaces will always be stripped from this search as Module Names cannot have spaces.

Alternatively, leave the box empty and click Lookup to list all modules. The search boxes above / below the table can also be used. A list of modules with their descriptions and associated genes is shown. These gene lists can be copied from the table and directly pasted into the regex search boxes that accept Gene Names - spaces will be stripped for you.

Explore Cytokines

The cytokines data is preloaded and immediately available. For details of the ABCD dataset see above. This is a highly experimental tab and very slow to plot.

Plot Cytokines

Use the menus to select some cytokines, vaccines and days. The order in which you enter the selections is respected in the plots, except for days on a continuous X axis.
Once you have selected some vaccines, days and cytokines you can click Plot to create plot, and to re-draw if you change any options.
This plot will take a long time if more than a few panels are plotted.

  • FI ~ Conc. Use the radio buttons to switch between fold increase (from day 0) or actual concentration values when plotting.FI may give some wide error bars if a few outliers have big changes.
  • Lines ~ Boxplot ~ Violin. Select which type of plot:
    Boxplot is the usual box-whiskers (IQR), Violin is a frequency distribution. They both offer the option to Crop Y, which zooms the plot to the IQR for the whole dataset - may be useful when extreme outliers squash the scale.
    Lines plots the mean of the cytokine~treatment combination on each day. Further options appear when selected:
    • Points adds a point to the lines.
    • No Error ~ Ribbon ~ Bars : SEM plotted as ribbon or error bars
  • Select • |----  to plot a horizontal line at Y = 1 or 0 (depends on whether log transformed).
  • Exclude Zero Values. The dataset already includes missing values (as NAs) where there was no data or "OOR" entered, or where a fold-increase could not be calculated (when day 0 was 0 or missing). It also contains many real zero values, where the concentration is entered as 0. To plot just the kinetics of the non-zero values, select this option and the data are filtered to remove zeros before means/medians/densities calculated.
  • Log transforms. Use the radio buttons to plot untransformed Y, or various log transforms. ln1p plots ln(1+Y) which is useful when there are zero values or Y << 1. For Boxplot and Violin plots the actual Y values are log transformed before any mean/medians are then calculated, so bear this in mind. For Line plots the raw Y values are also transformed before calculating the mean - unless the Means checkbox is selected, in which case the mean is calculated from untransformed data then log transformed (equivalent to plotting on a log scale), and the SE is set to 0 (as log transformation of variance is tricky). The raw data can always be downloaded from the data table to do your own transformations. Zero values are removed before transformation if Exclude Zero Y Values option is selected
  • Show N. This prints the number of available data points per day at the bottom of the plot, and the actual number used at the top. The top number may be reduced by missing data in the dataset, or when zero values have been omitted. The bottom number may also be reduced by data missing from the dataset (e.g. if one participant skipped a visit that reduces the group size for that day by 1). Useful if Exclude Zero Y Values selected.
  • Fixed Y. When selected the Y axis max-min range will be fixed for each cytokine across all treatments. Each cytokine will still have its own range, but the overall max - min will be imposed. This is relevant for placebo groups. If unselected, each panel has its own Y axis range.
  • Vac↓ Cyt→ creates panels grouped by vaccine: cytokines wrap in the columns, and then a new block starts with each vaccine treatment.
    Vac→Cyt↓ does the reverse, with panels grouped by cytokine: vaccine treatments wrap in the columns, and then a new block starts with each cytokine.
  • Monochrome - when selected all plots are in black and white. Otherwise I have pre-allocated fixed colours to the cytokines which do not change even when the number of cytokines plotted is changed.
  • A box to enter the maximum number of panels per row, and slider to control the height of the plot, offer some control over the image. Widening or narrowing the window also changes the aspect ratio.
The plot can be copied at low resolution by right-clicking, or a Hi-Res image downloaded by clicking the button. The data table can be downloaded by clicking the button.

Explore Cells

White blood cells data are not loaded automatically - click the button Load Cells Data to load an integrated dataset for trials ABC, D and D2.
Use the menus to select some cell types, vaccines and days. The order in which you enter the selections is respected in the plots, except for days on a continuous X axis.
Once you have selected some vaccines, days and cell types you can click Plot to create plot, and to re-draw if you change any options.
This plot will take a long time if more than a few panels are plotted.
  • Lines ~ Boxplot:Select which type of plot:
      • Boxplot is the usual box-whiskers (IQR).
      • Lines plots the mean of the cells type~treatment combination on each day. Further options appear when selected:
        • Points adds a point to the lines
        • SEM : SEM plotted as ribbon
        • Gridlines
  • Fold Change ~ Cell Count. Use the radio buttons to switch between plotting fold increase (from day 0) or actual cell values when plotting.
    When Cell Count Selected, the individual participant's cell counts are summarised on box plots, or the mean for the group plotted as a line chart.
    • Select • |----  to plot a horizontal line at Y = 1 when plotting Fold Change
  • Split Cells creates a new panel to plot each cell type under each vaccine sub-heading. If unselected then each cell type is plotted on one combined plot per vaccine: only really useful when Fold Change selected due to variations in normal ranges.
  • Free Y: When selected each plot panel has its own Y scale - useful only for looking at patterns. When unselected the Y scales are fixed by cell type across vaccines, but allowed to vary between cell types.
  • Summary Legend: produces one legend at the top of all plots, compared to ...
  • Plot Legends: where each plot panel has its own legend.
  • A box to enter the maximum number of panels per row, and slider to control the height of the plot, offer some control over the image. Widening or narrowing the window also changes the aspect ratio.
The plot can be copied at low resolution by right-clicking, or a Hi-Res image downloaded by clicking the button. The data table can be downloaded by clicking the button.
